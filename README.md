# 👕 Garment Offer Management System

A Python-based Garment Offer Management System developed using Object-Oriented Programming (OOP). This application helps garment stores manage customer purchases and validate free garment offers based on purchase price.

## 📌 Features

- Customer Management
- Purchase Price Calculation
- Free Garment Offer Validation
- Purchase History Storage
- File Handling
- Object-Oriented Programming (OOP)
- Modular Project Structure
- Exception Handling

## 📂 Project Structure

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Garment-Offer-System.git
```

2. Open the project folder

```bash
cd Garment-Offer-System
```

3. Run the program

```bash
python main.py
```
